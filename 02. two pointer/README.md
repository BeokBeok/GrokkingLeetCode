# Two Pointer 알고리즘이란?
- 두 개의 포인터 사용
- 처음에는 둘 다 시작점에 위치
- 마지막 포인터를 오른쪽으로 이동시키고, 조건에 만족하는 경우 마지막 포인터를 고정하고 시작 포인터를 오른쪽으로 이동 (반복)
